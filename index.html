What is DevOps?

DevOps is a set of practices that combines software development (Dev) and IT operations (Ops). It aims to shorten the development life cycle and provide continuous delivery with high software quality.

Tools of DevOps

  • Git & GitHub
  • Jenkins
  • Docker
  • Kubernetes
  • Ansible, Chef, Puppet
  • Nagios, Prometheus

Uses of DevOps

DevOps is used to increase deployment frequency, achieve faster time to market, lower failure rates, and improve team collaboration by automating infrastructure and workflows.

DevOps Life Cycle

  1. Plan
  2. Develop
  3. Build
  4. Test
  5. Release
  6. Deploy
  7. Operate
  8. Monitor
